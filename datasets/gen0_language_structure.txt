TAUTOLOGY AND CONTRADICTION

A tautology is always true. Regardless of the truth values of its parts.

P or not-P.

If P is true: true or false = true.
If P is false: false or true = true.

No escape. Always true.

A contradiction is always false. Regardless of the truth values of its parts.

P and not-P.

If P is true: true and false = false.
If P is false: false and true = false.

No escape. Always false.
